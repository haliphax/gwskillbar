# GW SkillBar

An online skill template code parsing/building tool for [Guild Wars Reforged][]

## Development

Cleanly install dependencies and run the development server:

```sh
npm ci
npm run dev
```

Skill and profession icons are not included in this repository. To download
them from the [Guild Wars Wiki][], use the `images` script (but please be
considerate of bandwidth usage):

```sh
npm run images
```

[guild wars reforged]: https://guildwars.com
[guild wars wiki]: https://wiki.guildwars.com
